# TokenSentinel Sandbox

Live interactive demo of TokenSentinel's leak interception engine.

## Deploy

### Vercel (recommended)

```bash
cd sandbox
npx vercel --prod
```

### GitHub Pages

Push to `main` — the `deploy-sandbox` workflow auto-deploys `sandbox/` when files in that directory change.

Enable Pages in your repo settings: Settings > Pages > Source = GitHub Actions.
