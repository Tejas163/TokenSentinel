API Reference

Complete reference for all TokenSentinel REST endpoints.

Authentication

All API endpoints (except the dashboard HTML page) require authentication via the Authorization header:

Authorization: Bearer <your-api-key>

The default development key is dev-key-123. Set via AUTH_API_KEY environment variable.

Dashboard Endpoints

Base URL: http://localhost:3001

GET/

HTML cost dashboard page. Does not require authentication.

GET/api/dashboard/cost-summary?period=24H

Aggregated cost breakdown by model.

ParameterTypeDescriptionValid Values
periodstringTime window for aggregation1H, 6H, 24H, 3D, 7D
{
  "total_cost": 1.234,
  "total_requests": 150,
  "total_tokens": 50000,
  "avg_tokens_per_request": 333.3,
  "models": [
    {"model": "gpt-4o-mini", "cost": 0.5, "requests": 100, "tokens": 30000, "pct": 40.5}
  ]
}
GET/api/dashboard/model-costs?period=24H

Per-model cost breakdown.

[
  {"model": "gpt-4o-mini", "cost": 0.5, "requests": 100, "input_tokens": 15000, "output_tokens": 15000}
]
GET/api/dashboard/events

Server-Sent Events stream for real-time cost and anomaly updates. The connection stays open and pushes events as they arrive.

data: {"type": "cost_event", "model": "gpt-4o-mini", "cost_cents": 0.042, "timestamp": "..."}

data: {"type": "anomaly", "model": "gpt-4o", "score": 4.2, "threshold": 3.0}

Admin Endpoints

GET/api/admin/health

Health check. Returns {"status": "ok"}.

GET/api/admin/anomalies?limit=10

Recent anomaly detection events.

ParameterTypeDescription
limitintNumber of events to return (default 10, max 100)

Team Management

GET/api/admin/teams

List all teams.

POST/api/admin/teams

Create a new team.

{
  "name": "engineering",
  "monthly_token_limit": 10000000
}
DELETE/api/admin/teams/{name}

Delete a team and its budget rules.

Budget Status

GET/api/admin/budget-status/{team_name}

Get current budget usage for a team.

{
  "team_name": "engineering",
  "monthly_token_limit": 10000000,
  "tokens_used": 4500000,
  "usage_pct": 45.0,
  "remaining": 5500000
}

Budget Rules

GET/api/admin/budget-rules

List all budget alert rules.

POST/api/admin/budget-rules

Create a budget alert rule.

{
  "team_name": "engineering",
  "threshold_pct": 80,
  "webhook_url": "https://hooks.slack.com/services/..."
}
DELETE/api/admin/budget-rules/{id}

Delete a budget rule by ID.
