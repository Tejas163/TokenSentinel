TokenSentinel

An enterprise-grade, distributed AI gateway that governs every token, every route, and every cost — autonomously.

Get Started Architecture GitHub
🛡️

Cost Governance

Every request records model + token usage with a CFO-ready dashboard. Real-time cost tracking, per-team budgets, and anomaly detection.

🔀

Intelligent Routing

Weighted random provider selection, autonomous model tier selection, and budget-aware rerouting when teams exceed limits.

⚡

Resilience

Circuit breakers, retry with exponential backoff + jitter, and in-memory route caching ensure high availability for your AI infrastructure.

📊

Observability

Real-time SSE dashboard, anomaly detection (3-sigma), Slack/Teams cost digests, and cost annotations on every proxied response.

🏢

Enterprise SDK

Spring Boot Starter SDK with Spring AI @Tool methods, Micrometer metrics, and auto-configuration for seamless Java integration.

🧬

Prompt Optimization

Evolutionary prompt optimization (EvoluNet-SLM) using mutation operators and LLM-based fitness scoring to discover high-performance prompts.

Polyglot Architecture

TokenSentinel is built with five languages, each chosen for its strengths:

ServiceLanguagePortResponsibility
rust-proxyRust3000TLS termination, API key auth, rate limiting, circuit breaker
go-routerGo8080Route resolution, provider selection, cost telemetry, budget check
cost-dashboardGo3001Web UI, aggregation API, anomaly detection, digest webhooks
erlang-monitorErlang—Health heartbeats, cost persistence
spring-boot-sdkJava—Enterprise SDK with Spring AI tools
evolunet-slmPython—Evolutionary prompt optimizer

Key Features

  • Sliding-window rate limiting via Redis
  • TLS termination & API key / JWT authentication
  • Weighted random provider load balancing
  • Autonomous model tier selection (20+ models)
  • Budget-aware routing per team
  • Cost annotations on every response
  • Real-time SSE cost dashboard
  • 3-sigma anomaly detection
  • Slack/Teams cost digests
  • Per-team budgets with monthly limits
  • Data retention and auto-pruning
  • Micrometer metrics integration
  • Evolutionary prompt optimization
